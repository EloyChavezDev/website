# L-Values vs R-Values

> L-value simply means an object that has an identifiable location in memory (i.e. having an address).

> What are R-values? an object that has no identifiable location in memory (i.e. having an address)

# Statements vs Expressions

- **Statements** are instructions that perform some action and do **not** return a value:
  ```rust
  let y = 6;
  ```
- **Expressions** evaluate to a resulting value (eg. macros & functions)

```rust
	let y ={
    let x = 3;
    x + 1
	};
```
