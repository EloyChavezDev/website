##### Introduction
Dans un monde où le travail hybride et à distance devient la norme, les entreprises recherchent des solutions qui offrent à la fois flexibilité, sécurité et efficacité. Windows 365 Link répond à ces besoins en tant que dispositif cloud PC, permettant aux utilisateurs de se connecter en toute sécurité à Windows 365 en quelques secondes. Ce dispositif révolutionne l'expérience de bureau virtuel en offrant une connexion rapide, une gestion simplifiée et une sécurité renforcée, tout en maximisant la productivité des utilisateurs.

---

##### Mise en place
Pour utiliser Windows 365 Link, les organisations doivent disposer de Windows 365 avec <u>Microsoft Intune</u> et <u>Microsoft Entra ID</u>, et répondre aux exigences existantes de Windows 365. Les éditions prises en charge incluent Windows 365 Enterprise, Windows 365 Frontline et Windows 365 Business.

- Acquérir Windows 365 Link : Contactez votre équipe de compte Microsoft pour obtenir des informations sur l'accès à Windows 365 Link.
- Configurer avec Microsoft Intune : Gérez Windows 365 Link dans la section des appareils de Microsoft Intune, aux côtés des autres PC que vous gérez.
- Connexion et Utilisation : Connectez Windows 365 Link à votre environnement Windows 365 pour une productivité instantanée et sécurisée.

---

##### Spécifications matérielles
- Processeur : Intel CPU (modèle non divulgué)
- Mémoire : 8GB RAM
- Stockage : 64GB (cloud-based, pas de stockage local)
- Ports (3 x USB-A 3.2, 1 x USB-C 3.2, 1 x HDMI, 1 x DisplayPort, 1 x Ethernet, 1 x Audio jack)
- Connectivité : Wi-Fi 6E et Bluetooth 5.3
- Affichage : 2 écrans 4K

##### Avantages
- Productivité maximisée : Windows 365 Link permet une connexion rapide et sécurisée à Windows 365, offrant une expérience utilisateur réactive et fluide. Les utilisateurs peuvent se connecter en quelques secondes et reprendre leur travail là où ils l'ont laissé.
- Sécurité renforcée : Le dispositif est conçu sans données locales ni applications, et sans utilisateurs administrateurs locaux, réduisant ainsi la surface d'attaque. L'authentification sans mot de passe via Microsoft Entra ID ajoute une couche de sécurité supplémentaire.
- Compatibilité étendue : Windows 365 Link prend en charge une large gamme de périphériques filaires et sans fil, y compris les moniteurs, les dispositifs audio, les claviers, les souris, les caméras et les dispositifs de stockage.
- Gestion efficace : Les administrateurs peuvent configurer et gérer Windows 365 Link rapidement et efficacement via Microsoft Intune. Cela permet une gestion centralisée et simplifiée des appareils, réduisant ainsi la complexité et les coûts opérationnels.
- Durabilité : Le dispositif contient au moins 50 % de contenu recyclé et est conçu pour être durable et réparable. Il est également certifié ENERGY STAR, contribuant ainsi à la durabilité environnementale.
- Flexibilité : Permet aux employés de travailler de n'importe où avec un accès sécurisé à leurs ressources.
